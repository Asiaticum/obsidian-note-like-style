# Note-like Style

Obsidianのノートに、note.comのような行間やスタイルを適用するプラグインです。
読みやすさを重視したスペーシングと、カスタマイズ可能な設定を提供します。

## 特徴

- **note.com風のスタイル**: 行間や段落の間隔を広めに取ることで、読みやすいレイアウトを実現します。
- **詳細なカスタマイズ**: 行の高さ、段落間のスペース、見出しのマージンなどを細かく調整できます。
- **適用範囲の指定**: 特定のフォルダやファイルのみにスタイルを適用することが可能です。
- **モード選択**: 編集モード（Live Preview）と閲覧モード（Reading View）それぞれへの適用を切り替えられます。
- **多言語対応**: 設定画面は日本語と英語に対応しています。

## 設定

設定画面から以下の項目をカスタマイズできます。

- **言語 (Language)**: 設定画面の表示言語（日本語 / English）。
- **行間 (Line Height)**: テキストの行の高さ。
- **段落間スペース (Paragraph Spacing)**: 段落間のスペースの大きさ (em単位)。
- **見出しの上マージン (Heading Top Margin)**: 見出しの上のスペース (em単位)。
- **見出しの下マージン (Heading Bottom Margin)**: 見出しの下のスペース (em単位)。
- **テキストカラー (Text Color)**: テキストの色を指定。
- **フォントファミリー (Font Family)**: 使用するフォントファミリー。
- **適用範囲 (Included Paths)**: スタイルを適用するファイルまたはフォルダのパスを指定。空欄の場合は全てに適用されます。
- **編集モードに適用**: Live Previewモードへの適用の有無。
- **閲覧モードに適用**: Reading Viewモードへの適用の有無。

## インストール

### 手動インストール

1. 最新のリリースから `main.js`, `manifest.json`, `styles.css` をダウンロードします。
2. Obsidianの保管庫の `.obsidian/plugins/note-like-style` フォルダにファイルを配置します。
3. Obsidianの設定 > コミュニティプラグイン からプラグインを有効化します。
